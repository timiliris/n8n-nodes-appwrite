# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-13

### Added
- Initial release of n8n-nodes-appwrite-full
- Database operations:
  - Create Document
  - Get Document
  - Update Document
  - Delete Document
  - List Documents
- Storage operations:
  - Get File
  - Delete File
  - List Files
- Users operations:
  - Create User
  - Get User
  - Update User
  - Delete User
  - List Users
- Appwrite API credentials support
- Comprehensive documentation
- TypeScript support
- ESLint configuration
- Jest testing setup

### Features
- Support for Appwrite Cloud and self-hosted instances
- Proper error handling with continue-on-fail support
- Full TypeScript type safety
- Compatible with n8n 0.198.0+
- Compatible with Appwrite 1.4.x - 1.8.x+
